QlinicQueue+ is an experimental queue management system for small clinics.

The goal of this project is to explore how a clinic queue system can work with both **online ticket booking** and **on-site registration**, while keeping the queue flow simple for patients and staff.

This repository is currently in the **planning and early development stage**.

---

## Project Goal

Build a simple clinic queue system where:

- Patients can take a queue ticket online
- Walk-in patients can be registered by staff
- Both online and walk-in tickets share the same queue
- Patients can check their queue status and estimated waiting time
